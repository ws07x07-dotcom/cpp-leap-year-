# cpp-leap-year-
C++ Leap Year Calculator (if-else practice)  Simple console program that checks if a given year is a leap year. Follows complete leap year rules: - Divisible by 4 (except century years)   - Century years must be divisible by 400  Features: ✅ Console input/output ✅ Complete leap year logic (test: 1900, 2000, 2024, 2025) ✅ Beginner C++ project
